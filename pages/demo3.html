Demo3 - view all slides of uploaded file

File: uploaded file

Description:

View all slides of uploaded file in choosen div.

Code:

                
<input type="file" id="uploadFileInput" /> <br> 
<div id="slide-resolte-contaniner" ></div> 
<script> 
    $("#slide-resolte-contaniner").pptxToHtml({
        pptxFileUrl: "",
        fileInputId: "uploadFileInput", /*id of input tag*/ 
        slideMode: false,
        slidesScale: "50%",
        keyBoardShortCut: false 
    }); 
</script>

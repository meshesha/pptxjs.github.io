Demo1 - view all slides

File: Sample_demo1.pptx

Description:

View all slides of "Sample_demo1.pptx" file in choosen div.

Code:

              
<div id="slide-resolte-contaniner" ></div> 
<script> 
  $("#slide-resolte-contaniner").pptxToHtml({ 
    pptxFileUrl: "Sample_demo1.pptx", 
    slidesScale: "50%", 
    slideMode: false, 
    keyBoardShortCut: false 
  }); 
</script>
